# Publishing Runbook

## Objective

Turn a research memo into Kindle-ready and Notion-ready artifacts.

## Steps

1. Select canonical source Markdown from `reports/source/`.
2. Confirm the memo has metadata, summary, claims, evidence, and open questions.
3. Run Kindle export.
4. Run Notion export.
5. Check output files.
6. Add source links and citations if missing.
7. If publishing through an API, verify destination page/database before upload.

## Kindle Checklist

- Title is clear.
- Date and version are visible.
- Executive summary appears before details.
- Sections are short.
- Tables are avoided or simplified.
- Source list is at the end.
- File is in `outputs/kindle/`.

## Notion Checklist

- Metadata block is present.
- Tags are present.
- Key claims are separated from evidence.
- Open questions and next actions are explicit.
- File is in `outputs/notion/`.
