# Weekly Finding: 2026-07-13

Status: draft
Scope: Microsoft Copilot usage metering

## What Changed

Microsoft's Copilot support page (last updated April 2026, opened 2026-07-13) confirms Microsoft meters consumer/business AI usage through an "AI Credits" system:

- Individual Microsoft 365 plans (Personal, Family, Premium) include Copilot subject to AI Credit usage limits.
- Microsoft 365 Premium buys advanced AI features, agents (Tasks, Analyst, Researcher, Photos Agent in preview), and an extended usage allowance.
- Business Copilot Chat is included with an eligible Microsoft 365 license; Microsoft 365 Copilot is a paid add-on; agents are pay-as-you-go.
